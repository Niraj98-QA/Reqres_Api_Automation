Dependencies

rest-assured	Core library for sending HTTP requests and testing REST APIs

json-path	To extract and validate values from JSON responses easily

xml-path (optional)	To parse and validate XML responses if needed

hamcrest	Provides powerful and readable matchers for assertions

testng or junit	Test framework to organize and execute test cases

jackson-databind	For converting Java POJOs to JSON and vice versa (serialization/deserialization)

apache-poi (optional)	To read/write data from Excel files (for data-driven testing)

log4j (optional)	Logging framework for debugging and logs

extentreports (optional)	To generate detailed and interactive HTML reports

json-schema-validator	To validate API responses against a predefined JSON Schema
